# Traveling Salesman Problem (TSP) Solver 🚀

A Streamlit-based web application to solve the Traveling Salesman Problem using exact and heuristic methods.

## Features

* Manual coordinate input
* Distance matrix input
* CSV upload
* Brute force & Nearest Neighbor algorithms
* Route visualization
* Execution time tracking

## How to Run

```bash
pip install -r requirements.txt
streamlit run app.py
```

## Tech Stack

* Python
* Streamlit
* NumPy
* Pandas
* Plotly / Matplotlib
